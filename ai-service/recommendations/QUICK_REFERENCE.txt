## 🐍 Python API Quick Examples

```python
from recommendations.treatment_recommender import TreatmentRecommender

rec = TreatmentRecommender()

# Get simple summary
print(rec.get_simple_treatment("Apple/Apple scab"))

# Get full detailed treatment
print(rec.get_full_treatment_text("Apple/Apple scab"))

# Get precautions
precautions = rec.get_precautions_list("Apple/Apple scab")

# Get best practices
practices = rec.get_best_practices("Apple/Apple scab")

# Search
results = rec.search_treatments("powdery mildew")

# Get all diseases
all_diseases = rec.get_all_diseases()

# Get by crop
rice_diseases = rec.get_diseases_by_crop("Rice")

# Get by severity
severe = rec.get_diseases_by_severity("Severe")

# Get statistics
stats = rec.get_summary_stats()
```

## 📁 Disease ID Format

Always use: `Crop/Disease`

Examples:
- `Apple/Apple scab`
- `Rice/Bacterial leaf blight`
- `Tomato/Early blight`
- `Wheat/Powdery mildew`

## 🔑 Treatment Data Fields

Each treatment contains:
```
disease_name         → Common name
crop                → Affected crop
severity            → Light/Moderate/Severe
primary_treatment   → Main pesticide info
  ├── pesticide     → Chemical name
  ├── brand_name    → Commercial brands
  ├── dosage        → Amount & frequency
  └── application_method
alternative_treatments → Other options
precautions         → Safety warnings
best_practices      → Prevention tips
effectiveness       → Success rate %
cost_estimate       → Price range
```

## 🛠️ Troubleshooting

| Issue | Solution |
|-------|----------|
| Module not found | Ensure treatment_recommender.py is in recommendations/ |
| Disease ID not found | Use format "Crop/Disease" |
| API not loading | Check treatments.json path |
| Validation error | All required fields must be present |
